# VSCode-In-Folder-Launcher

A Application that helps you to open vs code in any folder.

#
### Usage
- Just download.
- Extract the files.
- Run `VSCode-Folder-Launcher.exe`
- Enter the directory in which you want to open VS Code.
- It'll be open in vs code.
